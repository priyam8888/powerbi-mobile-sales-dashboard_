# Power BI Mobile Sales Dashboard

This project is a Mobile Sales Analysis Dashboard created using Power BI.

## Features
- Total Sales Analysis
- Sales by City
- Monthly Quantity Trend
- Customer Ratings
- Payment Method Transactions
- Sales by Mobile Model

## Tools Used
- Power BI
- Data Visualization
- Data Analysis
